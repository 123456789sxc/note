linux常见命令

因为git bash是linux环境下的shell，现在我们学习一下Linux的常见命令

cd  目录名      切换到该目录

cd ..         回退到上一目录

pwd   显示当前路径

ls   列出当前目录下的所有文件

touch  文件名.扩展名    在当前目录下创建一个文件

rm    文件名.扩展名   删除当前目录下的这个文件

mkdir 文件夹名    在当前目录下创建一个文件夹

rm -r 目录名  删除当前目录下的文件夹

mv 文件名.扩展名   文件夹名    在当前目录下，将文件名移入到文件夹中。注意文件和文件夹都在当前目录下

reset 重新初始化终端/清屏

clear 清屏

history 查看历史命令

exit   退出
